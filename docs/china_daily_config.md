# 中国日报配置说明

```python
CHINA_DAILY_CONFIG = {
    "class": "collectors.china_daily.ChinaDaily",
    "config": {
      "column": "global"
    }
}
```

其中config可选，column取值为：

- home
- global
- xismoments
- bilingual
- popular
- china
- business
- opinion
- world
- tech
- culture
- life
- travel
- sports
- ...
